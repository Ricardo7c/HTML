A linguagem HTML

HTML (HyperText Markup Language) é a linguagem de marcação fundamental usada para criar páginas da web.
Ela estrutura o conteúdo, organizando textos, imagens, links e outros elementos, permitindo que os navegadores os exibam corretamente.
o HTML define a hierarquia e a semântica do conteúdo. Ele é a base da web e é essencial para a construção de sites,
proporcionando a estrutura para aplicar estilos e adicionar interatividade com CSS e JavaScript.
